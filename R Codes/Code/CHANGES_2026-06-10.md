# R Code Changes on June 10, 2026

This note documents the code changes made on June 10, 2026.

## 1. Re-enabled Model 4 in Theorem 1 2D Sieve-Variance Simulation

File changed:

- `TEST_Thm1_2D_Spline_SieveVar.R`

Change:

- Uncommented the `Model 4` specification in the `specs` list.

Reason:

- The paper reports Theorem 1 sieve-variance simulation results for Models 4--7.
- With Model 4 commented out, the script only ran Models 5--7 and could not reproduce the full reported table.

## 2. Fixed Trimmed Control/Treated Splits in Empirical Welfare Scripts

Files changed:

- `TEST_Emp_Welfare_Spline_PlugIn.R`
- `TEST_Emp_Welfare_Spline_SieveVar_Trim.R`

Change:

- Replaced trimmed sample splits based on `ind` with splits based on treatment status `D`.

Old pattern:

```r
df_trim.c <- subset(df_trim, ind == 0)
df_trim.t <- subset(df_trim, ind == 1)
```

New pattern:

```r
df_trim.c <- subset(df_trim, D == 0)
df_trim.t <- subset(df_trim, D == 1)
```

Reason:

- `ind` is the trimming/common-support indicator, not treatment status.
- Since `df_trim <- subset(df, ind == 1)`, all observations in `df_trim` have `ind == 1`.
- Therefore `subset(df_trim, ind == 0)` is empty, and `subset(df_trim, ind == 1)` is the entire trimmed sample.

## 3. Saved DML Simulation Summary in RData Format

File changed:

- `TEST_DML_1D_Spline.R`

Change:

- Added a save statement at the end of the script:

```r
save(final_df, res_mat, file = "sim_results_DML_1D_Spline.Rdata")
```

Reason:

- Other theorem simulation scripts already save `final_df` and the last `res_mat` to `.Rdata`.
- This makes the DML script consistent with the existing simulation-script pattern.

## 4. Made Empirical Scripts Save Both Cost Scenarios

Files changed:

- `TEST_Emp_Value_Spline_SieveVar_NoTrim.R`
- `TEST_Emp_Value_Spline_SieveVar_Trim.R`
- `TEST_Emp_Welfare_Spline_PlugIn.R`
- `TEST_Emp_Welfare_Spline_SieveVar_Notrim.R`
- `TEST_Emp_Welfare_Spline_SieveVar_Trim.R`

Change:

- Replaced the single hard-coded `cost <- TRUE` or `cost <- FALSE` run with:

```r
cost_values <- c(FALSE, TRUE)
reports_df  <- data.frame()
for (cost in cost_values) {
  ...
}
```

- Each empirical script now appends both no-cost and cost-adjusted results into `reports_df`.
- Each script saves the combined results to both `.csv` and `.Rdata` files under `results/`.

Reason:

- The paper reports both:
  - no treatment-cost results; and
  - results subtracting the $774 treatment-assignment cost.
- The scripts previously required manually toggling `cost` and rerunning.

- The simulation scripts other than `TEST_DML_1D_Spline.R` still follow their existing pattern of saving `final_df` and the last `res_mat` to `.Rdata`.
